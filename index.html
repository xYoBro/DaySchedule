Getting Started
Create a schedule
From the library, tap + New Schedule, give it a name, and start adding days and events.
Edit events
Click any event on the schedule to open its properties in the inspector panel on the right.
Click the header
Click the schedule header (title/logo area) to open Settings — change the title, logo, contact info, and audience groups.
Saving
Auto-save
Every change is automatically saved 2 seconds after you stop editing. Watch for Saved in the toolbar.
First-time setup
The app needs permission to save. When prompted, pick the data folder inside your DaySchedule directory. You only do this once per browser.
Versions
Use the Versions button to save named snapshots (e.g., “Draft for Review”). You can always restore a previous version — the current state is auto-backed up first.
Keyboard Shortcuts
Ctrl + SSave now
Ctrl + PPrint all days
Ctrl + ZUndo
Ctrl + Shift + ZRedo
EscClose panels
Tips
Duplicate a schedule
Right-click a schedule in the library to duplicate it. Great for starting next month from this month’s template.
Audience groups
Main groups get full-width bands on the schedule. Limited groups appear as concurrent side notes when they overlap with main events.
Use Chrome or Edge
Auto-save only works in Chrome and Edge. Other browsers fall back to manual download.
