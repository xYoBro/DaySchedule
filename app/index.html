FAQ
What file do I open?
Open the .schedule file. That is the workbook.
What is import for?
Use import only for old exports or outside data you want to bring into a workbook.
Can multiple people edit?
Safest answer: one person edits the workbook file at a time.
Audience vs Specific People?
Audience is the group. Specific People is for named people.
Where do I change the look?
Use Customize.
Where is Save?
Use Save .schedule from the editor save bar or More menu.
When should I save?
Save the .schedule file before closing or handing it off.
